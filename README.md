🌿 Plant-O-Phool – Plant E-commerce Frontend Prototype

This project is a fully responsive plant e-commerce website built using HTML, CSS, and JavaScript (no backend). It focuses on modern UI/UX design, smooth user interaction, and a clean nature-inspired theme.

🔗 Live Demo:
https://plant-o-phool.netlify.app/

🚀 Features Implemented:

* Responsive and mobile-first design
* Nature-inspired UI with light and dark theme toggle (persisted using localStorage)
* Homepage with hero section, categories, featured plants, and testimonials
* Product listing page with 10–15 plant items and filter functionality
* Product detail page with large image view and horizontal scroll gallery
* Cart system with add/remove items, quantity control, and localStorage persistence
* Checkout (payment) page with form validation (frontend only)
* Authentication UI (login/signup interface)
* Embedded map for location/address
* Admin dashboard with analytics (sales, views, CTR) using data visualizations
* Reusable navbar and footer across all pages

🛠️ Tech Stack:

* HTML5
* CSS3 (Flexbox, Grid, responsive design)
* JavaScript (DOM manipulation, localStorage, basic validation)

💡 Key Highlights:

* Clean and modern UI focused on user experience
* Simulated real-world e-commerce flow without backend
* Modular and scalable frontend structure
* Smooth animations and interactive components

📌 Future Improvements:
